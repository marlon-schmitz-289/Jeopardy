# jeopardy-game

The Vue 3 + Electron frontend for the Jeopardy app. See the [root README](../README.md) for full project documentation.

## Dev Setup

```sh
npm install
```

### Run in browser (Vite dev server)
```sh
npm run dev
```

### Run as desktop app (Electron)
```sh
npm run electron:dev
```

### Type check
```sh
npm run type-check
```

### Build for distribution
```sh
npm run electron:build
```

Output is written to `release/`.

## Recommended IDE

[VSCode](https://code.visualstudio.com/) or any JetBrains IDE with the [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) plugin for Vue 3 + TypeScript support.
